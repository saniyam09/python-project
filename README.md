# Alarm Clock App

## Requirements
Python 3.9+
pip

## Install & Run

pip install flask flask_sqlalchemy flask-apscheduler flask-cors werkzeug

# Run
python app.py

# Open
http://127.0.0.1:5000

## Features
- Register/Login
- SQLite
- Alarms per user
- APScheduler
- Light/Dark theme
- Snooze + stop
